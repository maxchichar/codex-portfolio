# MAXCHICHAR 5D Spatial Portfolio

A cinematic, room-based spatial portfolio built with Next.js 15, React 19, and React Three Fiber.

## Run

```bash
npm install
npm run dev
```

## Replace Avatar / Auction Photo

- Avatar texture file path: `public/assets/avatar-source.jpg`.
- Gallery placeholder file path: `public/assets/gallery-placeholder.jpg`.
- Runtime upload: use the "Gallery Auction Room Upload" panel to drop/choose any image.

## Key Features

- Cryptographic cinematic loader key generated per visit.
- Door-opening ritual with holographic welcome sequence.
- Spatial room graph with camera rail transitions.
- Command palette (`Cmd/Ctrl + K`) to fly to any room.
- Auction room with swappable art via file upload.
